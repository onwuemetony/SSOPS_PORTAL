# SSOPS_PORTAL


The SSOPS_PoRTAL is a suite of applications I have developed over the years during the course of my career in telecoms.
I developed them as automations to make work more efficient for myself and team mates.
The apps mostly improve on how we gathered performance metrics.
The aim is to reduce errors/discrepancies that accompany manual population of spreadsheets

I started out using winforms developed in VB.net and flat text files as database. Issues with
version control each time I did an update made me migrate into an on-premise full webapp.

I am now in the process of making it fully functional on Microsoft Azure
Webapp + AzureSQL database
